1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/nubula-site.git
   ```
2. Install dependencies:

   ```bash
   pnpm install
   ```
3. Configure your `.env` with Discord and site credentials.

   ```bash
   pnpm run dev
   ```
5. Open your site at `http://localhost:3000`.
